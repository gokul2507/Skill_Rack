BST - Leaf Node Count (Id-3743)

An array of N positive integers is passed as input. The program must form a binary search tree with these numbers. The first number 
(out of the N numbers passed as input) is the root node of the binary search tree. The program must print the count of leaf nodes 
present in the binary search tree.
Note: A node is a leaf node if it does not have a left or right child.

Input Format:
The first line contains N numbers, each separated by a space.

Output Format:
The first line contains the count of leaf nodes.

Boundary Conditions:
1 <= N <= 9999

Example Input/Output 1:
Input:
1 2 5 3 6 4

Output:
2

Explanation:
The leaf nodes are the nodes with the values 4 and 6.

Example Input/Output 2:
Input:
9 11 4 7 2 6 5

Output:
3

Max Execution Time Limit: 3000 millisecs
