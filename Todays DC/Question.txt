Compare Two Students
The program must accept the name, age and registration number of two students as the input. The program must print the string value 
"EQUAL" if the details of the two students are equal. Else the program must print "NOT EQUAL" as the output. Ignore the case of alphabets 
while comparing the name and the registration number of two students.

Your task is to define the class Student so that the program runs successfully.

Input:
Hector
21
20uec147

20UEC147

EQUAL
