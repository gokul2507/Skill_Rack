Isolate Interlaced Strings (Id-3077)


Two string values S1 and S2 are interlaced and passed as a single input string S. Given L1 which is the length of S1, print S1 and S2 
as the output.

Input Format:
The first line contains S.
The second line contains L1.

Output Format:
The first line contains S1.
The second line contains S2.

Boundary Conditions:
4 <= LENGTH(S) <= 100
1 <= LENGTH(S1) <= 99
1 <= LENGTH(S2) <= 99

Example Input/Output 1:
Input:
LBARZIYSK
4

Output:
LAZY
BRISK

Max Execution Time Limit: 5000 millisecs
