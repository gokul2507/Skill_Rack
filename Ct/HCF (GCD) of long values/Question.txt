HCF (GCD) of long values (Id-3063)

The program must accept two long values X, Y and print their HCF (GCD).

Input Format:
The first line contains X and Y separated by a space.

Output Format:
The first line contains the HCF (GCD) of X and Y.

Boundary Conditions:
1 <=  X, Y <= 999999999999

Example Input/Output 1:
Input:
20 30

Output:
10

Example Input/Output 2:
Input:
999999999999 151515151515

Output:
30303030303

Max Execution Time Limit: 2000 millisecs
