ProgramID- 13957

Power Banks & Mobiles
There are N power banks and M mobiles. A person wants to charge the mobiles as full as possible. The amount of electric charge 
remaining(in percentage) in each power bank and each mobile are passed as the input. The program must print the maximum number of 
mobiles than can be fully charged as the output.

Note: 1% of electric charge in power bank = 1% of electric charge in mobile.

Boundary Condition(s):
1 <= N, M <= 1000

Input Format:
The first line contains N.
The second line contains N integer values separated by a space.
The third line contains M.
The fourth line contains M integer values separated by a space.

Output Format:
The first line contains the maximum number of mobiles than can be fully charged.

Example Input/Output 1:
Input:
2
51 25
4
0 99 50 85

Output:
3

Explanation:
The 3 mobiles that can be fully charged are 2nd, 3rd and 4th.
So 3 is printed as the output.

Example Input/Output 2:
Input:
2
50 100
5
20 30 40 50 20

Output:
2

Max Execution Time Limit: 50 millisecs
